色のクラスの基本
Bootstrapではそれぞれの用途ごとに決まった色が設定されている。

各色クラスの基本

color description
primary 主要な情報、一時情報
secondary 副次的な情報、二時情報
success 成功したことを報告
info インフォメーション
warning 警告
danger 危険
light 明るい状態
dark 暗い状態

各色見本

text-色名 table-色名 bg-色名
primary table-primary bg-primary
secondary table-secondary bg-secondary
success table-success bg-success
info table-info bg-info
warning table-warning bg-warning
danger table-danger bg-danger
light ※背景色明示 table-light bg-light
dark table-dark ※文字色明示 bg-dark ※文字色明示

間隔の調整
margin,paddingの指定をする際に使う省略したクラス。
m,pとx,y,t,b,r,lは続けて記述し、ハイフンでつないで幅の指定を行う。
m marginの設定
p paddingの設定
x left,rightの設定
y top,bottomの設定
t topの設定
b bottomの設定
r rightの設定
l leftの設定
0 幅をゼロに指定
1 幅を0.25remに指定
2 幅を0.5remに指定
3 幅を1remに指定
4 幅を1.5remに指定
5 幅を3remに指定
auto 幅を自動調整
this is m-3 p-3 space.
this is m-3 px-3 space.
this is m-3 py-3 space.
サイズの調整
margin,padding以外にwidth,heightを調整するクラス。
w,hの後ろに、ハイフンでつないで幅の指定を行う。
ｗ widthの設定
ｈ heightの設定
25 25％に設定
50 50％に設定
75 75％に設定
100 100％に設定
auto autoに設定
