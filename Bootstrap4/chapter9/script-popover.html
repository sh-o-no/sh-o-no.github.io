ポップオーバーの表示
ポップオーバーはpopoverメソッドを使って設定される。
placement：位置（top,bottom,left,rightのいずれか）
title：ポップオーバーのタイトルテキスト
content：ポップオーバーのコンテンツ
■scriptの内容 
$('<selector>')// 要素を取得
    .popover({// ポップオーバーにする
        'placement': 'bottom', // 位置
        'title': 'This is Popover!',// コンテンツタイトル
        'content': 'This is popover sample content.'// コンテンツ
    });
フォーカスによる非表示
本来のポップオーバーはボタンを再度クリックしないと
ポップオーバーが消えなくなっている。
data-trigger属性にfocusを指定することで
ボタン以外の場所をクリックするだけでポップオーバーを非表示にできる。
■scriptの内容 
$('<selector>')// 要素を取得
    .popover({// ポップオーバーにする
        'placement': 'bottom', // 位置
        'trigger': 'focus', // ON/OFF表示
        'title': 'This is Popover!',// コンテンツタイトル
        'content': 'This is popover sample content.'// コンテンツ
    })
HTMLを利用したポップオーバーの表示
ポップオーバーの内容にHTMLを操作しなくてもscriptの操作のみで可能。
引数に指定するもので変数名が入る場合は「''」は不要である。
'html':trueを指定するとtitleやcontentにHTMLコードを記述出来るようになる。（「''」不要）
■scriptの内容 
(function () {// 即時関数を利用
    // 変数を定義
    let img = '<img id="popoverImg" src="../img/sample.jpg" alt="sample" style="width: 75px;">';
    let title = '<div>This is popover!</div>';
    let content = 'This is <b>popover</b> sample content.';
    let html = '<div class="row"><div class="col-4">' + img + '</div>' + '<div class="col-8 lead">' + content + '</div></div>';
    $('<selector>').popover({// 要素を取得、引数を指定し、popoverを生成
        'placement': 'top',// 位置
        'trigger': 'focus',// ON/OFF表示
        'title': title,// コンテンツタイトル
        'html': true,// htmlコードを設定可
        'content': $($.parseHTML(html)),// コンテンツ
        -------------------------------------------------------------------------
        // JQueryオブジェクトで渡さないと、imgのstyleが無視される（多分Bootstrapのバグ）
        'content':html, 本来はこれでいけるはず。
        -------------------------------------------------------------------------
    });
})();
