<div>タグをアラートにする
アラートは<div>を使って作成されている為
<div>を使って表示されているコンテンツを
後からスクリプトでアラートに変更することも可能ということである。
■scriptの内容 
$(function () {// DOM構造が解析された後に動く
    $('#alertのID').addClass('alert alert-info');// IDの要素に対してクラスを追加
    $('#alertのID').addClass('alert alert-warning');
    $('#alertのID').addClass('alert alert-danger');
});
・スクリプト未使用
this is info alert message.
this is warning alert message.
this is danger alert message.
・スクリプト使用
this is info alert message.
this is warning alert message.
this is danger alert message.
アラートを閉じる
アラートはスクリプトの操作で閉じることも可能である。
$('ID').alert('close');でアラートを閉じる。
■scriptの内容 
$(function () {// DOM構造が解析された後に動く
    let counter = 0;// alertにalert1.2.3と順番に番号がついてることとする。

        $('#btnにつけたID').click(function () {// 指定したIDのボタンがクリックされた時の処理
            counter ++;// counterに1ずつ足す
            $('#alert' + counter).alert('close');// alertにcounterを足したIDが指定されている要素をcloseする。
        });

});
this is info alert message.
this is warning alert message.
this is danger alert message.
アラートを生成する
スクリプトでタグそのものを生成することで
新しくアラートを作って表示させることが出来る。
■scriptの内容 
$(function () {// DOM構造が解析された後に動く

    let counter = 0;// alertにalert1.2.3と順番に番号がついてることとする。

        $('#ボタンに指定したID').click(function () {// 指定したIDのボタンがクリックされた時の処理
            $('#alertのID').append(// 指定の要素の子要素に挿入
                $('<div>')// divタグ
                .attr('role', 'alert')// 属性を指定
                .addClass('alert alert-primary alert-dismissible')// クラスを指定
                .text('No,' + ++counter + '.This is alert!')// テキストを指定
                .append(// divタグの子要素に挿入
                    $('<button>')// buttonタグ
                    .addClass('close')// クラスを指定
                    .attr('data-dismiss', 'alert')// 属性を指定
                    .append(// buttonタグの子要素に挿入
                        $('<span>')// spanタグ
                        .attr('aria-hidden', true)// 属性を指定
                        .text('✕')// テキストを指定
                    )
                )
            );
        });

});
Alert will appear here!
